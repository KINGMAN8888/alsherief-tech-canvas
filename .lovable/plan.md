

# Youssef AlSherief — Futuristic Portfolio Website

## Vision
A dark, cyberpunk-professional single-page portfolio that embodies "Hardware meets Software" — deep slate backgrounds, neon sky blue accents, subtle gold circuit-line details, glassmorphism cards, and smooth scroll-triggered animations.

## Design System
- **Background:** Deep Slate (#0b1120 / #0f172a)
- **Primary Accent:** Neon Sky Blue (#38bdf8) — software/cloud side
- **Secondary Accent:** Gold/Copper (#cfb53b) — subtle hardware circuit lines/borders
- **Text:** White/Off-white
- **Typography:** Cairo (body/Arabic) + Rajdhani (English headings)
- **Effects:** Glassmorphism cards, glowing borders on hover, fade-in-on-scroll animations via Framer Motion

## Sections

### 1. Navbar (Fixed, Glassmorphic)
- Logo: "Youssef AlSherief" with a code/chip icon
- Navigation links: About, Services, Projects, Experience, Contact (smooth scroll)
- "Download CV" button with sky blue gradient
- Mobile hamburger menu

### 2. Hero Section
- Centered layout with animated "motherboard grid" background pattern (CSS/SVG)
- Headline: "Engineering the Future, One Line of Code & One Circuit at a Time."
- Sub-headline with role titles
- Two CTA buttons: "View My Work" (primary) + "Contact Me" (outline)

### 3. About Me
- Two-column layout: hexagonal photo frame (left) + bio text (right)
- Highlights: education in Russia & Egypt, trilingual (Arabic, English, Russian)
- Fade-in animation on scroll

### 4. Services (Hybrid Skillset)
- 4-card grid: Software Dev, Hardware & Networking, Cloud & DevOps, AI & Automation
- Glassmorphism cards with blue glow on hover
- Icons + brief descriptions for each category

### 5. Experience Timeline
- Vertical timeline with alternating sides (desktop), stacked (mobile)
- Roles: IT Manager → Assembly Technician → Digital Marketing Manager → Remote IT Manager → Freelancer
- Dates, company names, and key achievements
- Gold accent line connecting timeline nodes

### 6. Featured Projects
- Grid of project cards with image placeholders, titles, tech stack tags, and action links
- Projects: PC Master, Smart Answer AI Bot, Dom Dash Do, Al-Tawasul Platform, Python Hub
- Hover effects with blue glow

### 7. Education & Certifications
- Academic degrees and professional certifications displayed in a clean card/list layout
- Certification badges grouped by category (Networking, Cloud, Programming, etc.)

### 8. Technical Skills
- Visual skill tags/badges grouped by category (Languages, Frontend, Backend, DevOps, Hardware)
- Clean grid or tag-cloud style display

### 9. Contact & Footer
- Contact form (Name, Email, Message) with validation
- Contact info: email, phone, location
- Social links: LinkedIn, GitHub, Telegram
- Copyright © 2026

## Technical Approach
- Framer Motion for scroll-triggered animations and hover effects
- Google Fonts: Cairo + Rajdhani loaded via index.html
- Fully responsive, mobile-first design
- Lucide React icons throughout
- All content hardcoded from provided data
